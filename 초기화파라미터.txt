##파라미터보기
- show parameter 명령
- v$parameter 조회
  (예) SELECT name, value
       FROM v$parameter
       WHERE name LIKE '%이름%';

##파라미터 유형
-동적파라미터 : DB 재시작없이 수정가능
-정적파라미터 : 수정 시 DB 재시작 필요
-세션파라미터 : 특정 세션에서만 적용

##파라미터 파일
-Non Default Parameter가 기록되는 파일
-경로와이름 : $ORACLE_HOME/dbs/spfileDB이름
-파라미터 파일의 백업
CREATE pfile FROM spfile;
--> $ORACLE_HOME/dbs/initDB이름 파일 생성
--> spfile 문제 시 대신 데이터베이스 초기화
--> spfile 재생성

##파라미터 수정
ALTER SYSTEM SET 파라미터이름=값
[SCOPE=memory|spfile|both]
-SCOPE
--memory : 현재 인스턴스에만 적용
           DB 재시작 시 취소
--spfile : spfile에만 기록
           DB 재시작 시부터 적용
           주로 정적파라미터에 적용
--both : 현재 인스턴스부터 지속적으로 적용

ALTER SESSION SET 파라미터이름=값;
:세션종료 시 취소
